# Frontend

> Documentació en preparació. Veure [pla.md](../pla.md) per a l'estat complet.

El frontend del Visor 2030 és una **Single Page Application** construïda amb Vue 3 i Vite que visualitza indicadors ODS per a municipis, comarques i agrupacions territorials.

## Tecnologia

- **Vue 3** — Composition API, `<script setup>`
- **Vite 5** — build tool
- **Vue Router 4** — routing client-side
- **Tailwind CSS 3** — estils
- **D3.js 7** — visualitzacions de dades
